Adventskalendern™

En digital adventskalender på webben. Föräldrar kan fotografera ”gömda” presentpaket i hemma. Efteråt kan föräldrar registrera sig (+ användare) för att skapa en kalender och ladda upp gömställena till kalenderns olika luckor – något som ger lucköppnare sysselsättning under större delen av december månad. Projektet är byggt med backend i ASP.NET MVC /SQL-server, frontend i AngularJS / Bootstrap.
